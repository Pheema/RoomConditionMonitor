# RoomConditionMonitor

[CO2-mini](https://www.kk-custom.co.jp/emp/CO2-mini.html) と [Raspberry Pi](https://www.raspberrypi.org/) を利用して自宅環境の CO2 濃度を監視する。必要に応じて slack にメッセージを飛ばすようにした。
